Regional coverage map

Same Leaflet + ONS counties layer as the generic demo, with the regional preset and automatic framing so the map fits the highlighted counties (Gloucestershire, Wiltshire, Oxfordshire, Warwickshire, Swindon).

Preset

Script shared/js/uk-counties-regional.js wraps initNeoUkCountiesMap and defaults highlighted to:

  • Gloucestershire
  • Wiltshire
  • Oxfordshire
  • Warwickshire
  • Swindon

Call initNeoUkRegionalCountiesMap('map-id', …), or read NEO_UK_REGIONAL_HIGHLIGHTS if you need the list elsewhere. By default the view uses fitHighlightedBounds (see uk-counties-leaflet.js); set fitHighlightedBounds: false and supply center / zoom for a fixed opening view.

Generic component and API: uk-counties-map.html
