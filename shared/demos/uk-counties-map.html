UK counties map (Leaflet)

Shared component for demo sites: England counties / unitary authorities from ONS data (GeoJSON), with optional highlighted regions and hover labels.

Reuse in a site

Copy or reference shared/js/uk-counties-leaflet.js from this repo. Include Leaflet 1.9.x CSS and JS from unpkg (or your bundler), add a container div with an id, then call initNeoUkCountiesMap('your-id', options). See the script header in that file for the full API.

Regional highlight preset (same data, fixed county list): uk-counties-regional-map.html
